# 高度経済システム (Advanced Economy)

## 1. 物理レイヤーと金融レイヤーの分離

本作の経済は、移動に時間を要する「物理的なモノ」と、瞬時に移動する「デジタル通貨」の2層構造で成り立っています。

- **物理レイヤー（物流）**<br>
  アイテム（資源・製品）は物理的にその場所に存在しなければ取引できません。A星で売るには、A星の倉庫へ運ぶ必要があります。
- **金融レイヤー（通貨）**<br>
  通貨はデジタルデータであり、宇宙のどこにいても瞬時に送金・取引・両替が可能です。

## 2. 5つの基軸通貨

本作では、プレイスタイルや用途に合わせて5種類の通貨が流通しています。
それぞれの通貨は特定の活動によって供給され、特定の生産活動によって消費されます。これにより、異なる専門分野を持つプレイヤー同士の経済的な協力関係が生まれます。

### 1. スタンダード・クレジット ($CRED) - 【汎用・基礎】
- **概要**: 最も流通量が多い基軸通貨。初心者の生活の糧であり、経済の潤滑油。
- **入手方法**:
    - NPCからの基礎的な配送依頼（Tier 1資源など）。
    - ゴミ拾い、残骸回収などの雑務タスク。
- **用途（消費）**:
    - **Tier 1 資源（水、鉄鉱石など）** の採掘コスト。
    - 惑星の居住区維持費、税金。
    - 取引手数料。

### 2. インダストリアル・トークン ($IND) - 【重工業・建設】
- **概要**: 工業惑星ハブで重宝される、ハードウェア製造のための通貨。
- **入手方法**:
    - 大量の資源輸送ミッション。
    - 建設資材（鉄板、コンクリートなど）の納品タスク。
- **用途（消費）**:
    - **工場・施設** の建設。
    - **船体 (Hull)** の製造。
    - 採掘機や精製施設の製造。

### 3. テック・チップ ($TECH) - 【技術・研究】
- **概要**: 研究都市で発行される、精密機器とソフトウェアのための通貨。
- **入手方法**:
    - 未知の星系の探索データ提出。
    - 精密機器、レアメタルの輸送タスク。
- **用途（消費）**:
    - **モジュール**（エンジン、センサー、CPU）の製造。
    - 高度な設計図（ブループリント）のライセンス料。
    - 技術研究の加速。

### 4. ミリタリー・ボンド ($MIL) - 【軍事・治安】
- **概要**: 軍事政権や防衛隊が発行する軍事国債。戦う者のための通貨。
- **入手方法**:
    - 海賊討伐ミッション。
    - 重要輸送船団の護衛任務。
    - 特定空域のパトロールタスク。
- **用途（消費）**:
    - **武器**（レーザー、ミサイル、レールガン）の製造。
    - 弾薬、シールド発生装置の購入。
    - 傭兵の雇用契約。

### 5. ヴォイド・コイン ($VOID) - 【裏社会・リスク】
- **概要**: 支配者のいない無法地帯で使われる、匿名性の高い暗号通貨。
- **入手方法**:
    - 禁制品（ドラッグ、武器）の密輸。
    - 危険区域（放射能地帯など）の探索。
    - 企業へのスパイ活動、妨害工作。
- **用途（消費）**:
    - **違法改造パーツ**（オーバードライブ、ステルス）の製造。
    - クローキング装置、ハッキングツールの購入。
    - 海賊への通行料（みかじめ料）。

## 3. 取引市場システム

### ローカル市場（Spot Market）
- 各星系には、その地域の基軸通貨とのペアのみが存在します。
- 例: アルファ星（工業）では `鉄鉱石 / $IND` の板取引が可能。
- **裁定取引（Arbritrage）**<br>
  「$CREDで燃料を買い、$IND圏へ運んで売り、$INDで船体を買う」といった複雑なわらしべ長者的な貿易が可能です。

### グローバル為替市場（FX Market）
- 通貨同士の交換市場。場所を問わずアクセス可能。
- ペア: `$CRED / $IND`, `$IND / $TECH`, `$TECH / $MIL` など。
- 戦争が起きれば `$MIL` が高騰し、平和になれば `$IND` が上がるなど、情勢に連動した変動が発生します。

## 4. DeFi（分散型金融）要素

シミュレーション内に現代的な金融システム（DeFi）のメカニクスを導入しています。

### 流動性提供 (Liquidity Mining)
- 通貨ペア（例: `$IND / $TECH`）をプールに預けることで、他プレイヤーの両替手数料の一部を報酬として受け取れます。

### レンディング (Lending)
- **貸付**<br>
  資金が必要なプレイヤー（例: 戦争準備中のギルド）に通貨を貸し出し、利子を得る。
- **担保**<br>
  借り手は自身の所有する「艦船」や「ステーション権益」を担保（Collateral）として設定。
