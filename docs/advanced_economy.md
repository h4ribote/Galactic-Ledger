# 高度経済システム (Advanced Economy)

## 1. 物理レイヤーと金融レイヤーの分離

本作の経済は、移動に時間を要する「物理的なモノ」と、瞬時に移動する「デジタル通貨」の2層構造で成り立っています。

- **物理レイヤー（物流）**<br>
  アイテム（資源・製品）は物理的にその場所に存在しなければ取引できません。A星で売るには、A星の倉庫へ運ぶ必要があります。
- **金融レイヤー（通貨）**<br>
  通貨はデジタルデータであり、宇宙のどこにいても瞬時に送金・取引・両替が可能です。

## 2. 地域通貨とトークノミクス

市場カテゴリごとに独自のデジタル通貨が存在し、それぞれ異なるインセンティブ（用途）デザインがされています。

### $IND (Industrial Credit)
- **特徴**<br>
  巨大な資源市場を持つ「工業惑星ハブ」の基軸通貨。
- **用途**<br>
    *   基礎資源（鉄、燃料など）の購入。
    *   艦船の修理費、燃料補給費。
- **運用（ステーキング）**<br>
  銀行に預けることでサーバー全体の生産効率を上げ、その対価として金利を得る。

### $TECH (Tech Token)
- **特徴**<br>
  先進技術を持つ「研究都市ハブ」の通貨。
- **用途**<br>
    *   Tier 3（ハイテク）船体、モジュール、CPUの購入。
    *   設計図（ブループリント）のライセンス料。
- **運用（Burn）**<br>
  通貨を消費（Burn）することで技術開発速度を加速できる。消費需要によるデフレ（価値上昇）圧力がかかりやすい。

### $VOID (Void Coin)
- **特徴**<br>
  支配者のいない無法地帯の通貨。匿名性とリスクが高い。
- **用途**<br>
    *   海賊への通行料（みかじめ料）。
    *   禁制品、盗品の売買。
    *   傭兵への報酬支払い。
- **運用**<br>
  価格変動（ボラティリティ）が激しく、投機的な取引に利用される。

## 3. 取引市場システム

### ローカル市場（Spot Market）
- 各星系には、その地域の基軸通貨とのペアのみが存在します。
- 例: アルファ星（工業）では `鉄鉱石 / $IND` の板取引が可能。
- **裁定取引（Arbritrage）**<br>
  「A星で$INDで安く買い、B星へ運んで$TECHで高く売り、為替市場で$INDに戻す」といった複雑な貿易が可能。

### グローバル為替市場（FX Market）
- 通貨同士の交換市場。場所を問わずアクセス可能。
- ペア: `$IND / $TECH`, `$TECH / $VOID` など。
- プレイヤーは物流に関わらなくても、為替レートの変動を予測して利益を上げることができます（FXトレーダープレイ）。

## 4. DeFi（分散型金融）要素

シミュレーション内に現代的な金融システム（DeFi）のメカニクスを導入しています。

### 流動性提供 (Liquidity Mining)
- 通貨ペア（例: `$IND / $TECH`）をプールに預けることで、他プレイヤーの両替手数料の一部を報酬として受け取れます。
- これにより、生産や戦闘を行わない「純粋な投資家」としてのプレイスタイルが可能になります。

### レンディング (Lending)
- **貸付**<br>
  資金が必要なプレイヤー（例: 戦争準備中のギルド）に通貨を貸し出し、利子を得る。
- **担保**<br>
  借り手は自身の所有する「艦船」や「ステーション権益」を担保（Collateral）として設定。返済が滞ると自動的に所有権が貸主に移転します。
